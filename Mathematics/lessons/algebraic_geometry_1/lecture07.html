Theorem 1 Let $latex { X,Y }$ be varieties, assume $latex { Y\subset \mathbb{A}^m }$ be a closed affine variety. Then there is a bijection between morphisms $latex { X\rightarrow Y }$ and $latex { k }$-algebra homomorphisms $latex { A(Y)\rightarrow \mathcal{O}_X(X) }$:

$latex \displaystyle \begin{array}{cc} \{ \text{morphisms } X\rightarrow Y \} & \xrightarrow{bijection}\{ \text{homomorphisms }A(Y)\rightarrow \mathcal{O}_X(X) \} \\ \varphi & \xrightarrow{\qquad\quad} \varphi^\ast \end{array}$

Proof: $latex { \Rightarrow }$: Let $latex { \varphi :X\rightarrow Y }$ be a morphism, then $latex { \varphi^\ast: A(Y)\rightarrow \mathcal{O}_X(X) }$ is a $latex { k }$-algebra homomorphism. $latex { \Leftarrow }$: Let $latex {\phi:A(Y)\rightarrow \mathcal{O}_X(X) }$ be a $latex { k }$-algebraic homomorphism, let $latex { y_1,\dots,y_n\in A(Y) }$ be the coordinate functions. We set

$latex \displaystyle f_i=\phi(y_i)\in \mathcal{O}_X(X). $

Let $latex { \varphi=(f_1,\dots,f_m):X\rightarrow \mathbb{A}^m }$. This is a morphism from $latex { X }$ to $latex { Y }$. To see it is a morphism we have to show $latex { \varphi(X)\subset Y }$. Let $latex { h\in I(Y) }$, $latex { h\circ \varphi =h(f_1,\dots,f_m)=h(\phi(y_1),\dots,\phi(y_m))=\phi (h(y_1,\dots,y_m)) }$. The second equality is based on the homomorphic property of $latex { \phi }$, for example, if $latex { h(x_1,x_2)=x_1^2-x_2^3 }$, then $latex { h(\phi(y_1),\phi(y_2))=\phi(y_1)^2-\phi(y_2)^3= \phi(y_1^2)-\phi(y_2^3)=\phi(y_1^2-y_2^3)=\phi(h(y_1,y_2)) }$. So $latex { h(y_1,\dots,y_m)\in A(Y) }$, we choose an arbitrary element $latex { p=(a_1,\dots,a_m)\in Y }$, then $latex { h(y_1,\dots,y_m)(p)=h(a_1,\dots,a_m)=0 }$ because $latex { h\in I(Y) }$. So for arbitrary $latex { h\in I(Y) }$, we get $latex { h\circ\varphi=0 }$, it implies $latex { \varphi(X)\subset \mathop{\cap}_{h\in I(Y)} Z(h)= Y }$. $latex \Box$
Example 1 A bijective polynomial map need not to be an isomorphism. For example, let$latex { X=\mathbb{A}^1 }$, $latex { Y=Z(x_2^2-x_1^3) \subset \mathbb{A}^2 }$. Then

$latex \displaystyle \varphi=(t^2,t^3):X\rightarrow Y $

is a morphism and bijective and the inverse is

$latex \displaystyle \varphi^{-1}(a,b)=\left\lbrace \begin{matrix} \frac{b}{a} & \text{ if } a\neq 0\\ 0 & \text{ if } (a,b)=0 \end{matrix}\right. $

$latex { \varphi }$ is not an isomorphism($latex { \varphi^{-1} }$ is not a morphism). To show this we see the pull back:

$latex \displaystyle \varphi^\ast : A(Y)\rightarrow \mathcal{O}_X(X) $

where $latex { A(Y)=k[x_1,x_2]/\langle x_2^2-x_1^3\rangle }$ and $latex { A(X)=k[t] }$. $latex { \varphi^\ast }$ makes $latex { x_1\rightarrow t^2 }$ and $latex { x_2\rightarrow t^3 }$. Since $latex { \varphi^\ast }$ is not surjective(there is no element maps into $latex { t }$), $latex { \varphi^\ast }$ is not an isomorphism. By theorem 1 we know $latex { \varphi }$ is not an isomorphism. So bijective morphism is not necessary to be an isomorphism.

Proof: Let $latex { Z:=Z(\langle I(X),F\cdot x_{n+1}-1\rangle )\subset \mathbb{A}^{n+1} }$. We need to prove $latex { Z }$ is a closed subvariety of $latex { \mathbb{A}^{n+1} }$ isomorphic to $latex { X_F }$. As $latex { X_F }$ is irreducible, $latex { Z }$ is also irreducible. So $latex { Z }$ is closed variety of $latex { \mathbb{A}^{n+1} }$. On the other hand, the inverse of $latex { \varphi }$ is

$latex \displaystyle \varphi^{-1}=(x_1,\dots,x_n):Z\rightarrow X_F $

is a morphism, so $latex { \varphi }$ is an isomorphism. $latex \Box$

A map $latex { \varphi:X\rightarrow Y }$ is called a polynomial map if there exists homogeneous polynomials $latex { F_0,\dots,F_m\in k[x_0,\dots,x_n] }$ of the same degree with no common zero on $latex { X }$ s.t. $latex { \varphi(p)=[F_0(p),\dots,F_m(p)] }$, $latex { \forall p\in X }$, write $latex { \varphi=[F_0,\dots,F_m] }$.

$latex \displaystyle F_a:=F(1,x_1,\dots,x_n). $

Theorem 6 $latex { \varphi_i=(\frac{x_0}{x_i},\dots,\hat{\frac{x_i}{x_i}},\dots,\frac{x_n}{x_i}):U_i\rightarrow \mathbb{A}^n }$ is an isomorphism.

Corollary 1
  1. Every variety is isomorphic to a quasi-projective variety.
  2. Every variety has an open cover by affine varieties.

Proof: (1) Let $latex { X }$ be a variety, if $latex { X }$ is locally closed in $latex { \mathbb{P}^n }$, then it is a quasi-projective variety, so we only need to consider the condition in $latex { \mathbb{A}^n }$. Before we prove it, we need to prove a conclusion in topology.

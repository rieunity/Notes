Corollary 1 \noindent
  1. Let $latex { X }$ be a projective variety, then $latex { \mathcal{O}_X(X)=k }$, i.e., any regular functions on the whole of $latex { X }$ is constant;
  2. Let $latex { X }$ be a projective variety, $latex { Y }$ be an affine variety, then every morphism $latex { \varphi:X\rightarrow Y }$ maps $latex { X }$ to a point.

Proof: \noindent

  1. Let $latex { f\in\mathcal{O}_X(X) }$, then $latex { f:X\rightarrow \mathbb{A}^1 }$ is a morphism. Since $latex { X }$ is complete, we get $latex { f(X) }$ is closed in $latex { \mathbb{A}^1 }$. Hence $latex { f(X) }$ is a point or $latex { f(X)=\mathbb{A}^1 }$. Via embedding $latex { \mathbb{A}^1 }$ in $latex { \mathbb{P}^1 }$, $latex { f }$ is a morphism from $latex { X }$ to $latex { \mathbb{P}^1 }$. Since $latex { \mathbb{A}^1 }$ is not closed in $latex { \mathbb{P}^1 }$, $latex { f(X) }$ is a point.
  2. By the definition of affine variety, we know $latex { Y }$ is isomorphic to a closed subvariety of $latex { \mathbb{A}^n }$. Thus we can simply assume $latex { Y }$ is a closed subvariety of $latex { \mathbb{A}^n }$, then we can write the morphism as $latex { \varphi=(f_1,\dots,f_n) }$ with $latex { f_i\in \mathcal{O}_X(X) }$ for $latex { i=1,\dots,n }$. By (1) we just proved, we have $latex { f_i =a_i}$ for some $latex { a_i\in k }$. Thus $latex { \varphi(x)=(a_1,\dots,a_n)\in k^n }$ for arbitrary $latex { x\in X }$.
$latex \Box$
Remark 1 From this corollary we know that morphisms from projective varieties to affine varieties are quite boring. In the following, we introduce morphisms from projective varieties to projective varieties, called Veronese embedding.

Definition 2 (Veronese Embedding) Given fixed integer $latex { d,n>0 }$ and $latex { N:=\binom{n+d}{d}-1 }$, we construct the map

$latex \displaystyle \begin{array}{ccc} \nu_d:\mathbb{P}^n & \rightarrow & \mathbb{P}^N\\ {}[x_0,\dots,x_n] & \rightarrow & [ M_0,\dots,M_N ] \end{array} \ \ \ \ \ (1)$

where $latex { M_i\in k[x_0,\dots,x_n], 0\leq i\leq N }$ are all monomials of degree $latex { d }$. This is a morphism(note that monomials $latex { x_0^d,x_1^d,\dots,x_n^d }$ have no common zero), so $latex { \nu_d(\mathbb{P}^n) }$ is a closed subvariety of $latex { \mathbb{P}^N }$.

Proposition 3 $latex { \nu_d:\mathbb{P}^n\rightarrow \nu_d(\mathbb{P}^N) }$ is an isomorphism.

Proof: The open subsets $latex { U_i=\{ [a_0,\dots,a_n]\in \mathbb{P}^n |a_i\neq 0 \} }$ form an open cover of $latex { \mathbb{P}^n }$. For any monomial $latex { M_i }$ of degree $latex { d }$ in $latex { x_0,\dots,x_n }$, we denote the corresponding coordinate on $latex { \mathbb{P}^N }$ by $latex { z_{M_i} }$. $latex { \tilde{U_i}:= \nu_d(\mathbb{P}^n)\backslash Z(z_{M_i}) }$ are open subsets of $latex { \nu_d(\mathbb{P}^n) }$ and form an open cover of $latex { \nu_d(\mathbb{P}^n) }$. For every piece $latex { U_i }$ of $latex { \mathbb{P}^n }$, the map

$latex \displaystyle \nu_d|_{U_i}:U_i\rightarrow \tilde{U_i} $

is a morphism and the inverse of $latex { \nu_d|_{U_i} }$ is given by

$latex \displaystyle \nu_d^{-1}|_{\tilde{U_i}}=[z_{x_0^d},z_{x_0^{d-1}x_1},\dots,z_{x_0^{d-1}x_n}]. $

$latex \Box$
Example 1 \noindent
  1. Let $latex { n=1 }$, then we get the simplest Veronese embedding

    $latex \displaystyle \begin{array}{ccc} \nu_d:\mathbb{P}^1 \rightarrow & \mathbb{P}^d & {}\\ {}[x_0,x_1] \rightarrow & [x_0^d,x_0^{d-1}x_1,\dots,x_1^d] & {}. \end{array} $

    $latex { \nu_d(\mathbb{P}^1) }$ is called a rational normal curve.
  2. Let $latex { n=2,d=2 }$, we get the embedding

    $latex \displaystyle \begin{array}{ccc} \nu_2:\mathbb{P}^2\rightarrow & \mathbb{P}^5 & {} \end{array} $

$latex { \nu_2(\mathbb{P}^2) }$ is called Veronese surface.

Remark 2 Let $latex { F=\sum\limits_{i=0}^{N}a_iM_i }$ be a homogeneous polynomial of degree $latex { d }$ in $latex { x_0,\dots,x_n }$, $latex { X\subset\mathbb{P}^n }$ be a closed subvariety. Then we can get

$latex \displaystyle \nu_d(X\cap Z(F))=\nu_d(X)\cap Z(\sum\limits_i^{N}a_iz_{M_i}). \ \ \ \ \ (2)$

Thus we can use the isomorphism between $latex { \mathbb{P}^n }$ and $latex { \nu_d(\mathbb{P}^n)\subset \mathbb{P}^N }$ to reduce questions about intersections with hypersurfaces to intersections with hyperplanes.

Corollary 4 Let $latex { X\subset \mathbb{P}^n }$ be a projective variety, $latex { F\in k[x_0,\dots,x_n] }$ be a homogeneous polynomial of degree $latex { d>0 }$. Then we have the following properties

  1. $latex \Box$

Definition 6 (Rational Map) A rational map

$latex \displaystyle \varphi:X\dashrightarrow Y $

is an equivalence class $latex { \langle U,\varphi\rangle }$ of pairs $latex { (U,\varphi) }$ where $latex { \emptyset\neq U\subset X }$ is open and $latex { \varphi :U\rightarrow Y }$ is a morphism. Here $latex { (U,\varphi)\sim(V,\psi) }$ $latex { \Leftrightarrow }$ $latex { \varphi|_{U\cap V}=\psi|_{U\cap V} }$. We say $latex { \langle U,\varphi\rangle }$ is defined by $latex { (V,\psi) }$ if $latex { (V,\psi)\in \langle U,\varphi\rangle }$.

Remark 3 \noindent
  1. Let $latex { \varphi:X\dashrightarrow Y }$ be a rational map defined by $latex { (U,\varphi) }$, then $latex { \varphi }$ defines a morphism

    $latex \displaystyle \varphi:\text{dom}\varphi\rightarrow Y $

    with $latex { \text{dom}\varphi=\mathop{\cup}\limits_{(V,\psi)\sim(U,\varphi)} V }$ open in $latex { X }$. Define $latex { \varphi(p):=\psi(p) }$ if $latex { (V,\psi)\sim(U,\varphi) }$ for $latex { p\in V }$. We called $latex { \varphi }$ a rational map defined on $latex { \text{dom}\varphi }$.
  2. Rational maps $latex { f:X\dashrightarrow \mathbb{A}^1 }$ are equivalent to rational functions $latex { f\in K(X) }$. If $latex { f\in K(X) }$, then $latex { f\in \mathcal{O}_X(U) }$ is a rational map from $latex { X }$ to $latex { \mathbb{A}^1 }$. Conversely, if $latex { f }$ is a rational map from $latex { X }$ to $latex { Y }$, then it is a morphism from an open set $latex { U\subset X }$ to $latex { Y }$, hence $latex { f\in \mathcal{O}_X(U) }$ $latex { \Leftrightarrow }$ $latex { f\in K(X) }$.
  3. Let $latex { X }$ be a variety, $latex { f_i\in K(X) }$ for $latex { i=1,\dots,n }$. Then $latex { (f_1,\dots,f_n):X\dashrightarrow \mathbb{A}^n }$ is a rational map. It is a morphism from $latex { \mathop{\cap}\limits_{i=1}^{n}\text{dom}f_i\rightarrow \mathbb{A}^n }$.
  4. Let $latex { X\subset \mathbb{P}^n }$ be quasi-projective variety, $latex { f_0,\dots,f_m\in S(X) }$ not all $latex { 0 }$. Then $latex { [f_0,\dots,f_m]:X\dashrightarrow \mathbb{P}^m }$ is a rational map. We can also construct it in a different way: let $latex { F_0,\dots,F_m\in k[x_0,\dots,x_n] }$ be homogeneous of the same degree and not all of them are in $latex { I(X) }$, then $latex { [F_0,\dots,F_m]:X\dashrightarrow \mathbb{P}^m }$ is a rational map. For example, for $latex { p=[0,\dots,0,1]\in \mathbb{P}^n }$, the projection from $latex { p }$

    $latex \displaystyle \Pi_p=[x_0,\dots,x_{n-1}]:\mathbb{P}^n\rightarrow \mathbb{P}^{n-1} $

    is a rational map and a morphism from $latex { \text{dom}\Pi_p=\mathbb{P}^n\ \{ p \} }$.

Definition 7 A rational map

$latex \displaystyle \varphi:X\dashrightarrow Y $

is dominant if $latex { \varphi(\text{dom}\varphi) }$ is dense in $latex { Y }$. (This is equivalent to: $latex { \varphi }$ is dominant if $latex { \psi(V) }$ is dense in $latex { Y }$ for some pair $latex { (V,\psi)\in\langle U,\varphi\rangle }$.)

Remark 4 Let $latex { \varphi:X\dashrightarrow Y }$ be a dominant rational map defined on $latex { U\subset X }$, $latex { \psi:Y\dashrightarrow Z }$ be a rational map defined on $latex { V\subset Y }$, then $latex { U\cap \varphi^{-1}(V) }$ is a nonempty open subset of $latex { X }$. Then we get a composition $latex { \psi\circ\varphi:X\dashrightarrow Y }$. This allows us to define the pullback of rational functions. Let $latex { \varphi:X\dashrightarrow Y }$ be a dominant rational map, then $latex { \varphi^\ast f:=f\circ\varphi :X\dashrightarrow \mathbb{A}^1\in K(X)}$. It is easy to see $latex { \varphi^\ast:K(Y)\rightarrow K(X) }$ is a homomorphism.

Definition 8 A dominant rational map $latex { \varphi:X\dashrightarrow Y }$ is called a birational map if and only if there exists a dominant rational map $latex { \varphi^{-1}:Y\dashrightarrow X }$ such that $latex { \varphi^{-1}\circ \varphi=\text{id}_X }$ and $latex { \varphi\circ\varphi^{-1=\text{id}_Y} }$ are both rational maps. $latex { X }$ and $latex { Y }$ are called birational or birational equivalent if there exists a birational map $latex { \varphi:X\dashrightarrow Y }$.

Definition 9 A variety is called rational if it is birational to $latex { \mathbb{A}^n }$ for some $latex { n }$.

Example 2 \noindent

  2. $latex { C=Z(x_2^2-x_1^3)\subset \mathbb{A}^2 }$ is rational because the map $latex { (t^2,t^3): \mathbb{A}^1\rightarrow C }$ is birational.
  3. $latex { \mathbb{P}^n\times\mathbb{P}^m }$ is also rational because it contains an open subset $latex { U_{ij}\simeq \mathbb{A}^{n+m} }$.

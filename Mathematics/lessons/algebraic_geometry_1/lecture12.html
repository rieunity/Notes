Lemma 1 Let $latex { X,Y }$ be varieties and $latex { Y }$ be affine, let $latex { \varphi:X\rightarrow Y }$ is a morphism. Then $latex { \varphi(X) }$ is dense in $latex { Y }$ if and only if $latex { \varphi^\ast }$ is injective.

Proof: If $latex { \varphi }$ is not dense, then there exists a closed subset $latex { W\subsetneqq Y }$ and $latex { \varphi(X)\subset W }$. We can write $latex { W=Z(f_1,\dots,f_r) }$ for $latex { f_1,\dots,f_r\in A(X)\subset K(X) }$. By possibly taking a bigger $latex { W }$ we can write $latex { W=Z(f) }$ for some none zero element $latex { f\in A(X) }$. Now we find $latex { \varphi^\ast f=f\circ\varphi =0 }$, so $latex { \varphi^\ast }$ is not injective. Conversely, if some $latex { f\neq 0 \in K(X) }$ satisfies $latex { \varphi^\ast f=0 }$, then $latex { \varphi(X)\subset Z(f)\subsetneqq Y }$ is not dense. $latex \Box$

Theorem 2 Let $latex { X,Y }$ be varieties, there is a bijection

$latex \displaystyle \left\lbrace\begin{array}{c} \text{dominant rational maps} \\ \varphi:X \rightarrow Y \end{array}\right\rbrace\longleftrightarrow\left\lbrace\begin{array}{c} k-\text{algebra homomorphisms}\\ \varphi^\ast:K(Y)\rightarrow K(X) \end{array}\right\rbrace. $

In particular, $latex { X }$ and $latex { Y }$ are birational if and only if $latex { K(Y)\simeq K(X) }$.

Proof: We only need to construct the inverse map to $latex { \varphi\rightarrow \varphi^\ast }$. Let $latex { \phi:K(Y)\rightarrow K(X) }$ be a $latex { k }$-algebra homomorphism, we want to construct a rational map $latex { \varphi:X\rightarrow Y }$ such that $latex { \varphi^\ast=\phi }$. Replacing $latex { Y }$ by an open affine subset, we can now assume $latex { Y\subset \mathbb{A}^n }$ is closed. Let $latex { y_1,\dots,y_n\in A(Y) }$ be coordinate functions, then $latex { \phi(y_1),\dots,\phi(y_n)\in K(X) }$. We can find a nonempty open subset $latex { U\subset X }$ such that $latex { \phi(y_i)\in\mathcal{O}_X(U) }$ for all $latex { i=1,\dots,n }$. Then the map $latex { x\rightarrow (\phi(y_1)(x),\dots,\phi(y_n)(x)) }$ is a morphism from $latex { U }$ to $latex { Y }$. In fact, we restrict $latex { \phi }$ to $latex { A(Y) }$, then $latex { \phi }$ defines an injective homomorphism from $latex { A(Y) }$ to $latex { \mathcal{O}_X(U) }$. Then by theorem 4, we get a morphism $latex { \varphi:U\rightarrow Y }$ and by lemma 0 its image is dense in $latex { Y }$. Thus we find $latex { \varphi:X\dashrightarrow Y }$ which is dominant and $latex { \varphi^\ast=\phi }$. $latex \Box$

Corollary 3 Let $latex { X,Y }$ be varieties, the following statements are equivalent
  1. $latex { X,Y }$ are birational;
  2. $latex { X,Y }$ contain open subsets isomorphic to each other;
  3. $latex { K(X)\simeq K(Y) }$ as $latex { k }$-algebras.

Proof: (1) $latex { \Rightarrow }$ (2): Let $latex { \varphi:X\dashrightarrow Y }$ be a birational map with inverse $latex { \psi :Y\dashrightarrow X }$. We can check that $latex { \psi\circ\varphi }$ is the identity on $latex { U= \mathrm{dom}\varphi\cap \varphi^{-1}(\rm{dom}\psi) }$ and $latex { \varphi\circ\psi }$ is the identity on $latex { V= \mathrm{dom}\psi\cap\psi^{-1}(\mathrm{dom}\varphi) }$. Thus $latex { U }$ is isomorphic to $latex { V }$ by restrict $latex { \varphi }$ on $latex { U }$.

(2) $latex { \Rightarrow }$ (3): $latex { K(X)\simeq K(U) }$, $latex { K(Y)\simeq K(V) }$, and we know $latex { K(U)\simeq K(V) }$, thus $latex { K(X)\simeq K(Y) }$.

(3) $latex { \Rightarrow }$ (1): Just the conclusion of theorem 2. $latex \Box$

$latex \displaystyle B=b_1 A+\dots+b_n A:=\lbrace \sum b_ia_i|a_i\in A \rbrace. $

Definition 2 ($latex { R }$-module) Let $latex { R }$. An abelian group $latex { B }$ together with the composition $latex { \cdot:R\cdot B\rightarrow B }$ is called an $latex { R }$-module if and only if for arbitrary $latex { r,r_1,r_2\in R }$ and arbitrary $latex { b,b_1,b_2 \in B}$, the following conditions are satisfied

Definition 3 (Finitely generated module) An $latex { R }$-module $latex { B }$ is called finitely generated if there exist $latex { b_1,\dots,b_n\in B }$ such that

$latex \displaystyle B=b_1R+\cdot b_n R. $

Example 1

If $latex { A }$ and $latex { B }$ are $latex { k }$-algebras and $latex { A\subset B }$, then $latex { B }$ is also an $latex { A }$-module. By definition, it is equivalent between $latex { B }$ is a finite $latex { A }$-algebra and $latex { B }$ is a finitely generated $latex { A }$-module.

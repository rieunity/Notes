Proposition 2 Let $latex { X\subset \mathbb{A}^N }$ be an affine variety of dimension $latex { n }$ and $latex { F\in k[x_1,\dots,x_N]\backslash I(X) }$. If $latex { Z(F)\cap X\neq \emptyset }$, then $latex { \text{dim}(Z(F)\cap X)=n-1 }$.($latex { Z(F)\cap X }$ may not be irreducible).

Proof: We need to show for all irreducible components $latex { Y_i }$ of $latex { Z(F)\cap X }$, $latex { \text{dim} Y_i\leq n-1 }$ and there exists a component $latex { Y_j }$ with $latex { \text{dim}Y_j=n-1 }$(later we will show that all irreducible components have dimension $latex { n-1 }$). By Noether normalization theorem, there is a finite surjective morphism $latex { \Pi:X\rightarrow \mathbb{A}^n }$. Identify $latex { k[x_1,\dots,x_n] }$ with $latex { \Pi^\ast(k[x_1,\dots,x_n])\subset A(X) }$. Let $latex { \bar{F} }$ be the class of $latex { F }$ in $latex { A(X) }$, there exists a nonzero polynomial

$latex \displaystyle H=x_{n+1}^d + \sum\limits_{i=0}^{d-1}a_ix_{n+1}^i $

with $latex { a_i\in k[x_1,\dots,x_n] }$ such that $latex { H(x_1,\dots,x_n,\bar{F})=0 }$. Replacing $latex { H }$ by an irreducible factor if necessary, we can assume $latex { H }$ is irreducible. Let $latex { \varphi =(\Pi,F):X\rightarrow \mathbb{A}^{n+1} }$, $latex { \Pi = (x_1,\dots,x_n)\circ\varphi }$ is finite, thus $latex { \varphi }$ is finite. By definition $latex { \varphi (X)\subset Z(H) }$, then $latex { \varphi(X) }$ is a closed subvariety of dimension $latex { n }$ in $latex { Z(H) }$. Thus $latex { \varphi(X)=Z(H) }$, $latex { \varphi :X\rightarrow Z(H) }$ is a finite surjective morphism. By definition, $latex { Z(F)\cap X= \varphi^{-1}(Z(H,x_{n+1}))=\varphi^{-1}(Z(a_0)\times \lbrace 0\rbrace) }$, thus $latex { \text{dim}(Z(F)\cap X)=\text{dim} Z(a_0) }$ where $latex { a_0\in k[x_1,\dots,x_n] }$. If $latex { a_0 }$ is constant, then $latex { Z(F)\cap X=\emptyset }$, contradict with the condition, so drop it. Now we know $latex { a_0 }$ is nonconstant polynomial , hence $latex { \text{dim}Z(a_0)=n-1 }$. $latex \Box$
Theorem 3 Let $latex { X }$ be a variety, $latex { \emptyset\neq U\subset X }$, $latex { U }$ is an open subset of $latex { X }$. Then $latex { \text{dim}U=\text{dim}X }$.

Proof: Let $latex { U_0\subsetneqq U_1\subsetneqq \dots\subsetneqq U_n=U }$ be a chain in $latex { U }$, let $latex { X_i=\bar{U} }$ the closure of $latex { U_i }$ in $latex { X }$. By definition $latex { U_i=U\cap X_i }$, thus

$latex \displaystyle X_0\subsetneqq X_1\subsetneqq \dots \subsetneqq X_n=X $

is a chain in $latex { X }$, thus $latex { \text{dim}U\leq \text{dim}X }$. Let $latex { X_0\subsetneqq X_1\subsetneqq \dots\subsetneqq X_n=X }$ be a chain of largest length in $latex { X }$ and $latex { X_0=\lbrace x_0\rbrace }$ be a point, let $latex { W\subset X }$ be an open subset with $latex { x_0\in W }$. Then we set $latex { W_i=X_i\cap W }$ for all $latex { i }$. Since $latex { W_{i+1} }$ is dense in $latex { X_{i+1} }$, we have $latex { W_{i+1}\supsetneqq W_i }$ for all $latex { i }$. Thus $latex { W_0=\lbrace x_0\rbrace\subsetneqq W_1\subsetneqq \dots\subsetneqq W_n }$ is a chain in $latex { W }$, we get $latex { \text{dim}X=\text{dim}W }$. Thus we can replace $latex { X }$ by $latex { W }$ and $latex { U }$ by $latex { W\cap U }$. Now we reduce to the case $latex { X }$ is affine.
  1. If $latex { X=\mathbb{A}^n }$, let $latex { x_0 }$ be a point in $latex { U }$, $latex { X_i }$ be affine linear subspaces containing $latex { X_{i-1} }$ for all $latex { i }$. Put $latex { U_i=X_i\cap U }$, $latex { U_0\subsetneqq U_1\subsetneqq \dots\subsetneqq U_n }$ is a chain in $latex { U }$, then $latex { \text{dim}U=n=\text{dim}X }$.
  2. If $latex { X }$ is affine, there exists a finite surjective morphism $latex { \varphi:X\rightarrow \mathbb{A}^n }$. $latex { \varphi(X\backslash U)\subsetneqq \mathbb{A}^n }$ is closed, let $latex { f\in I(\varphi(X\backslash U)) }$ and $latex { V=\mathbb{A}^n\backslash Z(f) }$, $latex { V }$ is open and dense in $latex { \mathbb{A}^n }$, $latex { \text{dim}V=n }$. Let $latex { W=\varphi^{-1}(V)\subset X }$, then $latex { \varphi|_W:W\rightarrow V }$ is surjective and closed, thus $latex { \text{dim}W\geq \text{dim}V=n }$, but $latex { U\supset W }$, hence $latex { \text{dim}U\geq \text{dim}W\geq n }$.
$latex \Box$

Proposition 1 Let $latex { \varphi:X\rightarrow Y }$ be a morphism of varieties. Assume there exists a nonempty open subset $latex { U\subset Y }$ such that for all $latex { p\in U }$, $latex { \text{dim}(\varphi^{-1}(p))=n }$, then we have

$latex \displaystyle \text{dim}X=\text{dim}Y+n. $

Proof: We prove the statement by induction over $latex { \text{dim}Y }$. If $latex { Y }$ is a point, then it is trivial. If $latex { \text{dim}Y>0 }$, replacing $latex { Y }$ by an open affine subset $latex { V }$(i.e. replace $latex { Y }$ by $latex { Y\cap \mathbb{A}^k }$ for some $latex { k }$) and $latex { X }$ by an open affine subset of $latex { \varphi^{-1}(V) }$, we can assume $latex { X,Y }$ are both affine by theorem 11. In fact, $latex { X\subset \mathbb{A}^l }$ and $latex { Y\subset \mathbb{A}^m }$ for some $latex { l }$ and some $latex { m }$, are closed affine subvarieties. We can write $latex { \varphi=(F_1,\dots,F_m) }$ with $latex { F_i\in k[x_0,\dots,x_l] }$. Let $latex { g\in k[x_1,\dots,x_m] }$ such that $latex { \emptyset \neq Z(g)\cap Y\neq Y }$, then we set $latex { Y'=Z(g)\cap Y }$ and $latex { X'=\varphi^{-1}(Y') }$. By definition $latex { X'=X\cap Z(g(F_1,\dots,F_m)) }$ and it is not empty since its image $latex { Y' }$ is not empty. For any point $latex { p\in Y' }$, $latex { \varphi^{-1}(p) }$ in $latex { X }$ is also in $latex { X' }$, hence the dimension of fibres is still equal to $latex { n }$. By induction any irreducible component $latex { \tilde{X} }$ of $latex { X' }$ has the relation $latex { \text{dim}\tilde{X}=\text{dim}\tilde{Y}+n }$ with the corresponding $latex { \tilde{Y} }$ of $latex { Y' }$, hence $latex { \text{dim}X'=\text{dim}Y'+n }$. Since $latex { \text{dim}Y=\text{dim}Y'+1 }$ and $latex { \text{dim}X=\text{dim}X'+1 }$, we get $latex { \text{dim}X=\text{dim}Y+n }$. $latex \Box$

Definition 5 (Field Extension and Finitely generated Field Extension) Let $latex { K/k }$ be a field extension. For $latex { a_1,\dots,a_n\in K }$, denote $latex { k(a_1,\dots,a_n) }$ as the smallest subfield of $latex { K }$ containing $latex { k }$ and $latex { a_1,\dots,a_n }$. This is called field extension over $latex { k }$ by $latex { a_1,\dots,a_n }$. If there are $latex { a_1,\dots,a_n\in K }$ such that $latex { K=k(a_1,\dots,a_n) }$, we say $latex { K/k }$ is finitely generated.

Definition 6 (Algebraically Independent sets) Let $latex { K/k }$ be a finitely generated field extension, elements $latex { b_1,\dots,b_n\in K }$ are called algebraically independent over $latex { k }$ if there is no polynomial $latex { f\in k [x_1,\dots,x_n] }$ such that $latex { f(b_1,\dots,b_n)=0 }$. In particular, if $latex { b\in K }$ is algebraically independent over $latex { k }$, then $latex { b }$ is called transcendent over $latex { k }$.

Let $latex { k(x_1,\dots,x_n) }$ be a field of rational functions in $latex { n }$ indeterminants, it is easy to see $latex { k(b_1,\dots,b_n)\simeq k(x_1,\dots,x_n) }$ if $latex { b_1,\dots,b_n }$ are algebraically independent over $latex { k }$.

Definition 7 (Transcendence Basis) A maximal set of algebraically independent elements of $latex { K }$ over $latex { k }$ is called a transcendence basis.

Theorem 10 Let $latex { X }$ be a variety, then

$latex \displaystyle \text{dim} X=\text{trdeg} K(X)/k. $

Proof: By theorem 9, we can assume $latex { X=Z(F)\subset \mathbb{A}^n }$ is a hypersurface, $latex { F\in k[x_1,\dots,x_n] }$ is irreducible. We know $latex { \text{dim}X=n-1 }$. Then $latex { K(X)=k(y_1,\dots,y_n) }$, $latex { F(y_1,\dots,y_n)\in A(X)=k[x_1,\dots,x_n]/\langle F \rangle }$ and $latex { F(y_1,\dots,y_n)=0 }$ since $latex { X=Z(F) }$. To show the equality, we assume the last variable $latex { x_n }$ occurs in $latex { F }$, then we can get $latex { y_1,\dots,y_{n-1} }$ are algebraically independent. Otherwise, there exists a nonzero element $latex { G\in k[x_1,\dots,x_{n-1}] }$ with $latex { G(y_1,\dots,y_{n-1})=0 }$, then $latex { G(y_1,\dots,y_{n-1})\in \langle F\rangle }$.

Theorem 1 (Existence of a Primitive Element) Let $latex { k }$ be a field of characteristic $latex { 0 }$, $latex { L/k }$ is a finite field extension. Then $latex { \exists b\in L }$ such that $latex { L=k(b) }$.

Proof: } $latex { K(X) }$ is function field of $latex { X }$, let $latex { a_1,\dots,a_r }$ be a transcendence basis of $latex { K(X)/k }$, then $latex { K(X)/k(a_1,\dots,a_r) }$ is a finite algebraic extension. By theorem 1, there exists a primitive element $latex { b\in K(X) }$ such that $latex { K(X)=k(a_,\dots,a_r)(b) }$ and $latex { b }$ is algebraic over $latex { k(a_1,\dots,a_r) }$. Since $latex { b }$ is algebraic, there exists a polynomial $latex { F\in k(a_1,\dots,a_r)[x] }$ such that $latex { F(b)=0 }$. Write

$latex \displaystyle F=\sum\limits_{l}\frac{G_l(a_1,\dots,a_r)}{H_l(a_1,\dots,a_r)}x^l $

where $latex { G_l(x_1,\dots,x_r),H_l(x_1,\dots,x_r)\in k[x_1,\dots,x_r] }$. Now we view it as $latex { F(x_1,\dots,x_r,x)\in k(x_1,\dots,x_r,x) }$. Multiply $latex { F }$ by producting $latex { H_l }$'s and then divide it by the greatest common devisor of the new coefficients. We get $latex { f=\tilde{h}F\in k[x_1,\dots,x_r,x] }$, it is a primitive polynomial. Let $latex { Y=Z(f)\subset \mathbb{A}^{r+1} }$, it is a irreducible hypersurface. Then $latex { A(Y)=k[x_1,\dots,x_r,x]/\langle f\rangle }$, $latex { K(Y)=Q(k[x_1,\dots,x_r,x]/\langle f\rangle)\simeq Q(k[a_1,\dots,a_r,x]/\langle f(x) \rangle) }$ $latex { \simeq k(a_1,\dots,a_r)[x]/\langle f\rangle}$ $latex {\simeq k(a_1,\dots,a_r)(b) }$ $latex { \simeq K(X) }$. Then $latex { X }$ is birational to $latex { Y }$. $latex \Box$ This proof also implies $latex { \text{dim}X=\text{trdeg}K(X) }$.

— 1. Tangent Space, Singular And Nonsingular Points —

First we talk about cases of hypersurfaces in $latex { \mathbb{A}^n }$.
Definition 2 Let $latex { X=Z(f)\subset \mathbb{A}^n }$ be a hypersurface, assume $latex { I(X)=\langle f \rangle }$. A point $latex { p\in X }$ is called a singular point if and only if $latex { \frac{\partial f}{\partial x_i}(p)=0 }$ for $latex { i=1,\dots,n }$. Otherwise, $latex { p }$ is called a nonsingular point. Let

$latex \displaystyle X_{\mathrm{reg}}:=\lbrace p\in X|p\text{ is nonsingular}. \rbrace. $

$latex { X }$ is called smooth or nonsingular if and only if $latex { X=X_{\mathrm{reg}} }$.

Example 1
  1. $latex { X=Z(y-x^2) }$ is nonsingular.
  2. $latex { X=Z(y^2-x^2(x+1)) }$ has a singular point $latex { (0,0) }$.
  3. $latex { X=Z(y^2-x^3) }$ has a nonsingular point $latex { (0,0) }$.

Proposition 3 Let $latex { X\subset \mathbb{A}^n }$ be an irreducible hypersurface and $latex { \text{char}k=0 }$, then $latex { X_{\mathrm{reg}} }$ is open and dense in $latex { X }$.

Proof: Let $latex { F\in k[x_1,\dots,x_n] }$ be irreducible such that $latex { X=Z(F) }$, then $latex { I(X)=\langle F \rangle }$. Define

$latex \displaystyle X_{\mathrm{sing}}:=\lbrace \text{singular points of } X \rbrace. $

By definition $latex { X_{\mathrm{sing}}= Z(F,\frac{\partial F}{\partial x_1},\dots,\frac{\partial F}{\partial x_n}) \subset X}$ is closed. Since $latex { Z(F) }$ is irreducible, the only thing we have to show is $latex { X\neq X_{\mathrm{sing}} }$. Assume $latex { X=X_{\mathrm{sing}} }$ $latex { \Rightarrow }$ $latex { Z(\frac{\partial F}{\partial x_i})\supset X }$ $latex { \forall i=1,\dots,n }$ $latex { \Rightarrow }$ $latex { \frac{\partial F}{\partial x_i}=0 }$ $latex { \forall i=1,\dots,n }$. Since $latex { \text{char}k=0 }$, we get $latex { F }$ is constant, it si impossible. $latex \Box$ Second we talk about cases of affine algebraic sets.
Definition 4 Let $latex { f\in k[x_1,\dots,x_n] }$, $latex { p\in\mathbb{A}^n }$. The differential of $latex { f }$ at $latex { p }$ is defined as

$latex \displaystyle \text{d}_p f=\sum\limits_{i=1}^{n}\frac{\partial f}{\partial x_i}(p)x_i. $

Let $latex { X\subset \mathbb{A}^n }$ be an affine algebraic set, the tangent space to $latex { X }$ at $latex { p\in X }$ is defined as

$latex \displaystyle T_p(X)=Z(\mathrm{d}_p f| f\in I(X)). $

$latex { p\in X }$ is called nonsingular if

$latex \displaystyle \mathrm{dim}T_p(X)= \mathrm{dim}_pX $

where $latex { \mathrm{dim}_pX }$ is the maximu of dimensions of irreducible components of $latex { X }$ passing through $latex { p }$.

Remark 1 If $latex { I(X)=\langle f_1,\dots,f_r\rangle }$, then $latex { T_p(X)=Z(\mathrm{d}_pf_1,\dots,d_p f_r) }$. By definition, $latex { T_p(X)\subset Z(\mathrm{d}_pf_1,\dots,\mathrm{d}_pf_r) }$. If $latex { h\in I(X) }$, we can write $latex { h=\sum\limits_{i=1}^{r}h_if_i }$ with $latex { h_i\in k[x_1,\dots,x_n] }$. Using Leibniz rule we get

$latex \displaystyle \mathrm{d}_p h=\sum\limits_{i=1}^{r}(\mathrm{d}_ph_i\cdot f_i(p)+h_i(p)\cdot \mathrm{d}_pf_i). $

Since $latex { f_i(p)=0 }$, we get $latex { \mathrm{d}_p h \in \langle \mathrm{d}_pf_1,\dots,\mathrm{d}_pf_r \rangle }$. Hence $latex { T_p(X)=Z(\mathrm{d}_pf_1,\dots,d_p f_r) }$.

Example 2 If $latex { X=Z(F) \subset \mathbb{A}^n}$ and $latex { I(X)=\langle F \rangle }$, then $latex { T_p(X)=Z(\mathrm{d}_pF) }$ and $latex { \mathrm{d}_pF=\sum\limits_{i=1}^{n}\frac{\partial F}{\partial x_i}x_i }$. Let $latex { \frac{\partial F}{\partial x_i}(p)=0 }$ $latex { \forall i=1,\dots,n }$ for some point $latex { p\in X }$, $latex { \forall i=1,\dots,n }$, then $latex { T_p(X)=\mathbb{A}^n }$. Since $latex { \mathrm{dim}T_p(X)\neq \mathrm{dim}_pX }$, $latex { p }$ is a singular point. If $latex { \frac{\partial F}{\partial x_i}(p)\neq 0 }$ for some $latex { i }$, then $latex { \mathrm{dim}T_p(X)=n-1 }$ and $latex { p }$ is nonsingular.

Third we talk about cases of general affine varieties.

Definition 5 (Jacobian) Jacobian of $latex { f_1,\dots,f_r\in k[x_1,\dots,x_n] }$ is a matrix defined as

$latex \displaystyle J(f_1,\dots,f_r)=\left(\begin{matrix} \frac{\partial f_1}{\partial x_1} & \frac{\partial f_1}{\partial x_2} &\cdots &\frac{\partial f_1}{\partial x_n}\\ \frac{\partial f_2}{\partial x_1} & \frac{\partial f_2}{\partial x_2} & \cdots & \frac{\partial f_2}{\partial x_n}\\ \vdots & \vdots & \ddots & \vdots\\ \frac{\partial f_r}{\partial x_1} & \frac{\partial f_r}{\partial x_2} & \cdots & \frac{\partial f_r}{\partial x_n} \end{matrix}\right) . $

Definition 6 Let $latex { X\subset \mathbb{A}^n }$, $latex { Y\subset \mathbb{A}^m }$ be closed subvarieties. Let $latex { p\in X }$, $latex { q\in Y }$ and $latex { \varphi = (f_1,\dots,f_m):X\rightarrow Y }$ with $latex { f_i\in k[x_1,\dots,x_n] }$ for $latex { i=1,\dots,m }$. Assume $latex { \varphi(p)=q }$. The differential of $latex { \varphi }$ at $latex { p }$ is

$latex \displaystyle \mathrm{d}_p\varphi = (\mathrm{d}_pf_1,\dots,\mathrm{d}_pf_m). $

One can verify that $latex { \mathrm{d}_p\varphi }$ maps $latex { T_p(X) }$ into $latex { T_q(Y) }$. We can wirte $latex { \mathrm{d}_p\varphi = J(f_1,\dots,f_m)(p)\cdot x }$, but now we change the definition of $latex { \mathrm{d}_p\varphi }$ a little

$latex \displaystyle \mathrm{d}_p\varphi =J(f_1,\dots,f_m). $

By using this new definition, we get
Proposition 7
  1. $latex { \mathrm{d}_p\mathrm{Id}=\mathrm{Id} }$.
  2. $latex { \mathrm{d}_p(\psi\circ\varphi)=\mathrm{d}_{\varphi(p)}\psi \circ\mathrm{d}_p\varphi }$.

At last we talk about tengent spaces for general varieties.

Definition 8 Let $latex { X }$ be a variety, $latex { p\in X }$ be a point. The tangent space $latex { T_p(X) }$ is

$latex \displaystyle T_p(X):=(\mathfrak{m}_p/\mathfrak{m}_p^2)^{\ast} $

where $latex { \mathfrak{m}_p }$ is the maximal ideal of the local ring $latex { \mathcal{O}_{X,p} }$, the symbol $latex { \ast }$ denotes the dual of vector space. In other words,

$latex \displaystyle T_p(X)=\lbrace k \text{ linear maps }\nu: \mathfrak{m}_p/\mathfrak{m}_p^2\rightarrow k \rbrace $

or

$latex \displaystyle T_p(X)=\lbrace k \text{ linear maps }\nu: \mathfrak{m}_p\rightarrow k \text{ with } \nu|_{\mathfrak{m}_p^2}=0 \rbrace $

$latex { p\in X }$ is called nonsingular if $latex { \mathrm{dim}T_p(X)=\mathrm{dim}X }$. Similarly we have definition of $latex { X_{\mathrm{sing}} }$ and $latex { X_{\mathrm{reg}}.}$ If $latex { X=X_{\mathrm{reg}} }$, $latex { X }$ is called nonsingular or regular.
